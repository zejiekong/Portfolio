Robotic Arm

As a student who is passionate about robotics, it was only natural for me to take up the MA4825 robotics module offered as part of my school curriculum. What's more exciting is that this module has a project that requires students to build a robotic arm system which targets a specific problem. In my team, I was involved in developing the software of the robot arm in which I leveraged on tools like ROS and Python. My personal contributions in this area are:
1.) Google Speech-to-Text API Integration 2.) Motor Control Using SDK and Inverse Kinematics Implementation Overall, this was a very fun and exciting project. In fact, it was the first time that I collaborated with somebody on a ROS-based project. The feeling of watching your code bring a robot to alive is truly amazing.
