Sugarcane Machine Design

Basically, groups of 3 have to each design a sugar cane machine, using the concepts and guidlines taught in the module. The project really tested our ability to make logical and reasonable mechanical design decisions. One thing to note about this project was that there was no official leader assigned which was pretty rare for a group project. This project was one of the most time-consuming projects that I have done. Despite that, I really enjoyed the whole learning process, from calculating the minimum specification for each machine elements to 3D-modelling sugar cane machine in SOLIDWORKS. Besides that, it also taught me that there is always an explanation behind every design decisions. This is particularly important if the end goal is to minimise cost while still able to fulfill certain requirements. Through this module, I also understood the importance of having safety factor on top of calculated specifications. Highlights of this project: Although we didn't have a leader, everybody was very active and responsive. One of the best teams I have worked with. If not, the best 😁
