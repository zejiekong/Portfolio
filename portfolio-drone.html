DJITello Drone

MLDA NTU Junior Robotics Engineer Project

Using ROS and Aruco Detection to develop a simple vision-based navigation system for DJITello drone. Due to many hardware and software issues, the project was not able to progress as expected. However, this project served as a great hands-on practice for using ROS. In the demo video, the drone first takes off and ascend until it detects the aruco marker. Next, it moves towards the marker slowly and lands at a distance from the marker. The distance is estimated using an aruco pose estimator script. The ROS architecture can be seen in the second image. Currently, the source code resides in the MLDA-RMUA github private repository.
